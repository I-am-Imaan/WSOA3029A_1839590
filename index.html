About Us

Diabetic Drop is a website aims to debunk stereotypes commonly
associated with diabetes as well as provides basic diabetes knowledge, in order
to create a better understanding of the illness to both diabetics and non-diabetics.
Together we can create a world that is safer and more understanding toward diabetics.
